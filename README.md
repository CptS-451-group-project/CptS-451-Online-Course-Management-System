# [Online Course Management System]
> Our project is a Online Course Management System that will modernize student enrollment and faculty administration.

## 📌 Project Overview
The online course management system will allows student's to browse, enroll, and drop courses. The professor will be able to manage the course such as removing the student if needed. The administrator will manage the entire application such as viewing, approving, and rejecting enrollments.
